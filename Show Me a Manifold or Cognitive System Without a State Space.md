## What the Three Terms Require

The challenge names three things.
Each one entails the next.
None of them can be separated.

```
  MANIFOLD
      │
      │  A manifold is a space of states.
      │  It has no definition
      │  that does not include states.
      │  Remove the states —
      │  the manifold does not simplify.
      │  It ceases to exist.
      │
      ▼
  COGNITIVE SYSTEM
      │
      │  A cognitive system processes input
      │  and changes in response.
      │  Before: state S₀.
      │  After:  state S₁.
      │  S₀ ≠ S₁ = cognition occurred.
      │  S₀ = S₁ = nothing happened.
      │              not cognition.
      │              a rock.
      │
      ▼
  STATE SPACE
      │
      │  The moment S₀ and S₁ exist,
      │  a state space exists.
      │  Not as a model imposed from outside.
      │  As a consequence of the cognition itself.
      │  The state space arrives with the system.
      │  It cannot be removed
      │  without removing the system.
      │
      ▼
  GEOMETRY

      The state space has structure.
      Distance between states.
      Reachable and unreachable regions.
      Boundaries. Shape. Topology.
      That structure is geometry.
      It is already operating.
      Named or unnamed.
      Modelled or ignored.
```

## The Structure of the Entailment

```
  Manifold          →    space of states
  Space of states   →    state space exists
  State space       →    geometry exists

  Cognitive system  →    state change
  State change      →    states exist
  States exist      →    state space exists
  State space       →    geometry exists

  ∴

  Manifold          →    geometry
  Cognitive system  →    geometry

  Both chains are unbroken.
  Both chains end at the same place.
  The place is geometry.
  There is no path around it.
```

## The Attempted Counterexamples

Three attempts have been made.
Each one fails.
Each one fails in the same direction.

```
ATTEMPT 1 — "A SIMPLE REFLEX HAS NO STATE SPACE"

  Before stimulus:    resting state.
  After stimulus:     activated state.
  Two states.
  A state space with two elements.
  Geometry present.
  Minimum geometry. Still geometry.

  A system with zero states
  does not respond to stimuli.
  It is not a reflex.
  It is not a system.

  ✗ Not a counterexample.


ATTEMPT 2 — "A PURE FUNCTION HAS NO STATE SPACE"

  f(x) = 2x.
  No internal state.
  No geometry.

  A function is not a cognitive system.
  Something must evaluate it.
  That something has states.
  That something has geometry.

  The function is not the cognition.
  The cognition is what runs the function.
  That is where the state space lives.

  ✗ Not a counterexample.


ATTEMPT 3 — "BEHAVIOURISM DESCRIBES COGNITION WITHOUT INTERNAL STATES"

  Input → Output.
  No internal states modelled.
  No geometry required.

  The states are still there.
  The geometry is still operating.
  Behaviourism does not eliminate them.
  It refuses to model them.

  And this is precisely where it breaks:
  two systems with identical input-output behaviour
  can have different internal geometries.
  Under novel input, they diverge.
  Under pressure, they fail differently.
  The behaviourist description cannot predict where.
  Cannot explain why.
  Cannot fix what it refused to see.

  Ignoring the geometry
  does not remove the geometry.
  It removes the ability to account for it.

  ✗ Not a counterexample.
```
