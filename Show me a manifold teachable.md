## What a State Is

A **state** is a snapshot. The complete configuration of a system at one moment in time.

```
  BEFORE                          AFTER

  ┌─────────────────┐             ┌─────────────────┐
  │                 │             │                 │
  │   System X      │  [input]    │   System X      │
  │   at moment 0   │ ─────────▶  │   at moment 1   │
  │                 │             │                 │
  │   S₀            │             │   S₁            │
  │                 │             │                 │
  └─────────────────┘             └─────────────────┘

  S₀ = the system before.
  S₁ = the system after.
  S₀ ≠ S₁ = something happened. Input was processed.
  S₀ = S₁ = nothing happened. Not cognition. A rock.
```

The moment a system has a before and an after, it has two states. Two states is already a state space.

## What a State Space Is

A **state space** is the complete set of all configurations a system could possibly be in — every before, every after, everything reachable, everything not reachable.

```
  A STATE SPACE — DRAWN:

  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │   •   •       •                   •                     │
  │         •           •       •                           │
  │   •           •                       •   •             │
  │         •                   •                   •       │
  │   •   •       •       •           •       •             │
  │                   •       •   •                   •     │
  │         •   •         •               •       •         │
  │                                                          │
  │   Each • = one possible configuration of the system.    │
  │   The full set of • = the state space.                  │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

```
  BUT NOT ALL STATES ARE EQUAL:

  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │   REACHABLE FROM S₀:          UNREACHABLE FROM S₀:      │
  │                                                          │
  │   ████████████████            ░░░░░░░░░░░░░░░           │
  │   ██  •───•───•  ██           ░░  •   •   • ░░          │
  │   ██  │   │   │  ██           ░░              ░░         │
  │   ██  •───•───•  ██           ░░  •   •   • ░░          │
  │   ██      │      ██           ░░░░░░░░░░░░░░░           │
  │   ██     S₀      ██                                      │
  │   ████████████████                                       │
  │                                                          │
  │   ████ = states X can reach.                            │

  │                                                          │

  │   It has structure.                                      │
  │   That structure is geometry.                            │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

## What a Manifold Is

A **manifold** is a state space with shape — curved, structured, uneven terrain rather than a flat grid.

```
  FLAT GRID (not a manifold):       MANIFOLD (curved terrain):

  •───•───•───•───•                    •
  │   │   │   │   │                  ╱   ╲
  •───•───•───•───•                •       •
  │   │   │   │   │               ╱ ╲     ╱ ╲
  •───•───•───•───•              •     •─•     •
  │   │   │   │   │              ╲   ╱     ╲   ╱
  •───•───•───•───•                •           •

  All states equidistant.          States are near or far
  No curvature.                    depending on structure.
  No valleys. No ridges.           Has valleys. Has ridges.
  No geometry.                     Has geometry.
  Nothing interesting.             Everything interesting.
```

```
  THE MANIFOLD OF A LANGUAGE MODEL — SIDE VIEW:

  High
    │
    │   \        ridge        /\       ridge      /
    │    \      (separatrix) /  \    (separatrix)/
    │     \                /    \               /
    │      \      ★       /      \      ★      /
    │       \   basin    /        \   basin   /
    │        \          /          \         /
    │         \________/            \_______/
    │
    └───────────────────────────────────────────  state space

  The manifold is curved.
  It has valleys (basins).
  It has ridges (separatrices).
  It has attractors (★) at the valley bottoms.
  It is not flat.
  It is not uniform.
  Its shape determines everything
  the system can and cannot do.
```
